2024 H2 Chemistry P1 Q1 — Atomic structure Solution

Atomic structure 2024 H2 Chemistry · Paper 1 · Q1

Question

Cation X has the same number of neutrons as protons. Cation X and an atom of neon have the same number of electrons. What is cation X?
  1. A\({}^{16}\text{O}^{2-}\)
  2. B\({}^{19}\text{F}^{-}\)
  3. C\({}^{23}\text{Na}^{+}\)
  4. D\({}^{24}\text{Mg}^{2+}\)
Correct answer: D

Video Solution

Explanation

Since X is a cation, option A and B are wrong. Neon has 10 electrons. Cation X has 10 electrons. Na has 11p 12n, thus not correct. Mg has 12p 12n. Correct.
