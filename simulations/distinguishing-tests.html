How to use

Pick a chemical test by dragging reagent + condition chips into the Test zone. Then assign the correct observation chip to each compound. Answer is correct when both observations match the database and are different from each other.

Set questions

For 3- and 4-compound questions, you may need 2 (or 3) different tests. Every compound must end up with a unique combination of observations.

Hints

Click Hint to glow-highlight one valid reagent or condition you're missing in the active test, or one observation chip you haven't placed.

Common tests

  • Br₂(aq), KMnO₄/H⁺, K₂Cr₂O₇/H⁺ — alkenes / alcohols / aldehydes
  • 2,4-DNPH / Tollens' / Fehling's / I₂/NaOH — carbonyls
  • NaHCO₃ / Na / FeCl₃ — acids vs phenols vs alcohols
  • NaOH(aq) heat then HNO₃ then AgNO₃ — halogenoalkanes
  • Litmus, PCl₅ — bases / OH-containing
